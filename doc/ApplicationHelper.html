    Description¶ ↑

    Inserts values into model database and returns the id

    If values already exist in the database, return the id of the tuple with same values.

    Example call¶ ↑

    insert_into Instructor, first_name: "Harvey", last_name "Siy"
    # File app/helpers/application_helper.rb, line 16
def insert_into(model, values)
  return model.where(values).select(:id)[0].id if model.where(values).count > 0

  model.new(values).save
  model.last.id
end
